# Flutter + Firebase Image Sharing 
<img src="https://img.shields.io/badge/Flutter-1.17-blue"> <img src="https://img.shields.io/badge/Dart-lightgrey">

## Covered topics
- [x] Upload and compress image with jpg
- [x] Select image from camera roll or camera App
- [x] Photo tagging with Geolocation
- [x] Firebase Cloud Firestore
- [x] Firebase Cloud Functions 
- [x] Inside app notifications 
- [x] Like / unlike posts 
- [x] Users search


## Installation
- Run flutter clean
- Then flutter packages get
- Create and download Google Service plist from firebase
